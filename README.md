# PGADMIN4 
## Инструкция
1. Клонируем репозиторий в папку www:
```sh
git clone https://github.com/littlePiggy/pgadmin 
```
2. Заходим в папку:
```sh
cd pgadmin
```
3. Выполняем команду:
```sh
docker-compose up -d
```
___
### Доступы
- Доступ к pgadmin по ссылке: http://localhost
- Пользователь: admin
- Пароль: littlePiggy
___
### Про дампы
- Все сделанные дампы лежат по пути: pgadmin/pgadmin/data
___
